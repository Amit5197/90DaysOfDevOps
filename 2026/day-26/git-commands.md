### Task 6: Useful `gh` Tricks
Explore and try these — add the ones you find useful to your `git-commands.md`:
1. `gh api` — make raw GitHub API calls from the terminal
2. `gh gist` — create and manage GitHub Gists
3. `gh release` — create and manage releases
4. `gh alias` — create shortcuts for commands you use often
5. `gh search repos` — search GitHub repos from the terminal

Command,Best for...,User Intent
gh api,Custom automation,"""I need to do something the CLI doesn't support yet."""
gh gist,Portability,"""I have a single file I need to share or store."""
gh release,Distribution,"""I am ready to ship this version to users."""
gh alias,Efficiency,"""I'm tired of typing this long command every hour."""
gh search,Research,"""I need to find a tool or library to solve a problem."""
